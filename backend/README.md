Grocery Ecommerce that is built using NodeJS(NestJS), GraphQL, PostgreSQL & Sequelize
## Include a .env
```
DB_DEV_USER=
DB_DEV_PASSWORD=
DB_DEV_DATABASE=
DB_DEV_HOST=
DB_DEV_PORT=

JWT_SECRET=
JWT_EXPIRE_TIME=
```

## Install all the required modules

```
npm i 
```

## Start the server

```
npm start
```
